# tvoja mami

## How to run
### Naredi venv (ce zelis, priporocam)
```
python -m venv venv 
venv\Scripts\activate
```
### Install requirements
```
pip install -r requirements.txt
```

### Zalaufi backend
```
uvicorn main:app --reload --port 8000
```

### Zalaufi frontend
```
desn klik na index.html -> Open with live server
```
